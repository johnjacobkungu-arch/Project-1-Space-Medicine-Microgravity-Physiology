# Project-2-Space-Medicine-Microgravity-Physiology
Prolonged exposure to microgravity induces profound physiological adaptations, notably cardiovascular deconditioning. This analysis explores longitudinal changes in systolic blood pressure (SBP) among astronauts, comparing pre-flight baselines to in-flight and post-flight metrics. Using a mixed-effects modeling approach in R, we quantify the extent of baroreceptor-mediated hemodynamic shifts to better understand orthostatic intolerance upon return to Earth gravity (1G).
